Control Flow and Loops

Control flow is the order of the flow of things being executed in the program. Imagine you have a fast-flowing river and you chuck in a stick then a kayak and o and on with random things. The control flow controls what comes out and over the waterfall first. Loops are a block of code that runs around I loops. You decide the specific parameters of this. Think of it like a round conveyor belt. you put all the items in the control flow order and then decide how many times you want them to go around in a loop.

What's the DOM

DOM stands for Document Object Model (DOM). This allows you to manipulate the elements and code in a page on the web through the inspect command. You can mess around with anything in there and it can also help you locate a particular part of code if you want that!

Functions

A function is an action you want to create For example you could create a funciton that changed the color inside a dot from green to blue by double clicking it.
